1-day Advanced React & Redux Training

April 28th, 2018. From 9:30am to 6pm. 1 Fore St Ave, London EC2Y 9DT

On completion of the 1-day React & Redux training each student will:

  • Understand functional programming applied to React
  • Be able to write tests for complex React applications
  • Comprehend the best performance practices in React
  • Understand the best patterns for building reusable UIs & functionality in React

Curriculum

  • Functional programming concepts applied to React
    • Data and behaviour
    • Data in, data out
    • Composition
  • Advanced React patterns: Reuse functionality across components
    • Higher-Order Components (HOCs)
    • Declarative composition using the Render Props (AKA function as children)
  • Advanced React patterns II: Create more reusable components
    • Compound Components, dynamically flow data between components
    • Patterns and use cases using "context"
  • Functional programming
    • Composing functions
    • Currying, thunks and higher-order functions
  • Advanced Redux:
    • Introducing the Redux middleware
    • Exercise, build your own Redux middleware
    • Thunks. Actions that dispatch actions
    • Exercise, implement loading indicators using thunks
