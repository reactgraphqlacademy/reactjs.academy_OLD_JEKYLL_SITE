  • How much JavaScript do developers need to know to attend this training?

    To be a good React developer, developers must be good JavaScript developers. We recommend to have some experience with modern ES6 JavaScript before attending any of our React training. That being said, we always start our training with an ES6 workshop to make sure you have a good JavaScript foundation to leverage React.

  • Do developers need to have some experience in other JavaScript frameworks to attend this training?

    React is a paradigm shift, it takes a Functional approach rather than Object Oriented. It is declarative instead of imperative. Most traditional frameworks don't operate in these new paradigms, therefore knowking other front-end frameworks is not an advantage.
