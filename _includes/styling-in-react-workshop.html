Styling in React workshop

On completion of the evening workshop each student will:

  • Understand different approaches to styling in React
  • Understand styled-components and be able to use them

Evening Styling in React Workshop Curriculum

  • SASS implementation
    • add SASS support to your Create React App
    • understand the use cases of this approach
  • styled-components
    • add styled-components to your application
    • refactor your components into styled-components
